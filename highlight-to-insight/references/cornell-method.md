# 康奈尔笔记法 (Cornell Note-Taking System)

## 起源

由康奈尔大学教育系教授 Walter Pauk 在 1940 年代提出，在其畅销书《How to Study in College》中系统阐述。

## 五个步骤 (5 R's)

| 步骤 | 英文 | 说明 |
|------|------|------|
| 记录 | **R**ecord | 在笔记栏记录课堂/阅读中的核心内容 |
| 简化 | **R**educe | 课后尽快将笔记栏内容提炼为线索栏的关键词和问题 |
| 背诵 | **R**ecite | 遮住笔记栏，只看线索栏尝试回忆完整内容 |
| 反思 | **R**eflect | 思考内容之间的联系，与已有知识建立关联 |
| 复习 | **R**eview | 定期回顾笔记，强化长期记忆 |

## 页面布局

```
┌──────────────────────────────────────────┐
│  标题行（课程/书名、日期、章节）          │
├─────────────────┬────────────────────────┤
│  线索栏 (Cue)    │  笔记栏 (Notes)         │
│  ~6cm           │  ~15cm                 │
│                 │                        │
│  · 关键词        │  · 主要观点             │
│  · 核心问题      │  · 重要细节             │
│  · 公式/术语     │  · 图表数据             │
│  · 记忆提示      │  · 例证引用             │
│                 │                        │
│  （占 1/3）      │  （占 2/3）             │
├─────────────────┴────────────────────────┤
│  总结栏 (Summary)                        │
│  ~5-7cm 高度                            │
│  · 用自己话概括本页内容（1-3 句）          │
└──────────────────────────────────────────┘
```

## 线索栏编写原则

好的线索栏内容应该：
- 能触发回忆——看到关键词就能想起右侧的详细内容
- 可测试——问题应该引导主动回忆，而非被动再认
- 简洁——每个条目不超过一行

**示例对比**：

| 差 | 好 |
|:---|:---|
| 光合作用（太模糊） | **光合作用**：光能→化学能，发生在叶绿体 |
| 发生了什么？（无指向） | ❓ 光反应和暗反应的本质区别是什么？ |
| 重点（没有信息量） | **卡尔文循环**：CO₂固定的三个阶段 |

## 笔记栏编写原则

- 用自己的话重新表述，而非逐字抄录
- 使用列表、缩进、符号组织层次
- 每个要点控制在 2-4 句
- 在要点间留白，方便后续补充
- 标注不确定或有疑问的内容

## 总结栏编写原则

- 必须在 24 小时内撰写（记忆新鲜时）
- 用自己的话概括，测试是否真正理解
- 回答：「如果只能记住本页的一个观点，应该是什么？」
- 可包含与已有知识的联系

## 为什么康奈尔笔记有效

1. **生成效应**：自己生成内容比被动阅读记忆更深
2. **间隔重复**：线索栏设计天然支持自我测试和间隔复习
3. **双重编码**：同时使用语义（文字）和空间（布局）编码信息
4. **元认知**：写总结迫使你判断自己是否真正理解
