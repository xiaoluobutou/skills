# 微信读书导出格式参考

## 常见导出格式

### 1. Markdown 格式（浏览器扩展导出）

```
# 《书名》
> 作者：作者名
> 导出时间：2024/1/1 00:00:00

◆ 第一章 标题

>> 划线内容第一句...
>> 划线内容第二句...

📝 想法：用户对这段划线的想法

◆ 第二章 标题

>> 划线内容...
```

**关键特征**：
- 书名以 `# ` 开头
- 作者行以 `> 作者：` 开头
- 章节以 `◆ ` 开头
- 划线以 `>> ` 开头
- 想法/笔记以 `📝 想法：` 开头
- 可能包含 `🖍️`（马克笔）、`〰️`（波浪线）、`➖`（直线）等标记类型

### 2. 另一种 Markdown 格式（WeRead Export 扩展）

```markdown
书名
作者
N 个笔记
阅读周期:
* 阅读时长：X 小时
* 开始时间：DD/MM/YYYY
* 结束时间：DD/MM/YYYY

> 划线内容...

📝 想法：用户笔记
```

### 3. JSON 格式

```json
{
  "bookName": "书名",
  "author": "作者",
  "notes": [
    {
      "chapter": "第一章",
      "highlight": "划线原文",
      "note": "用户想法（可为空）",
      "type": "mark",
      "time": "2024-07-25"
    }
  ]
}
```

## 关键字段映射

| 字段 | 康奈尔笔记用途 |
|------|--------------|
| 书名 (bookName) | 笔记标题 |
| 作者 (author) | 标题栏信息 |
| 章节 (chapter) | 笔记分组单元 |
| 划线 (highlight) | 笔记栏素材 |
| 想法/笔记 (note) | AI 理解上下文 + `💭` 标注 |
| 划线类型 (type) | 重要性参考（马克笔 > 波浪线 > 直线） |
| 时间 (time) | 日期标注 |
